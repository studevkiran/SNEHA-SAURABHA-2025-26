## Requirements

- **Format**: JPG or PNG
- **Max Size**: 5MB
- **Recommended Dimensions**: 800x600 pixels
- **Aspect Ratio**: 1.91:1 (horizontal) or 1:1 (square)
- **Must be**: High quality, clear, professional

## How It Will Be Used

This logo will appear as:
1. **WhatsApp message header** (top of the message)
2. **Confirmation page header** (in browser)

## URL

Once uploaded, it will be accessible at:
- `https://sneha2026.vercel.app/images/event-logo.jpg`

This URL is used in the Infobip WhatsApp template.
